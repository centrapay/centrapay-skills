# Centrapay Skills

Skills for building Centrapay integrations with AI coding agents.

## Available Skills

| Skill | Description |
|---|---|
| [`centrapay-e-commerce-integration`](./centrapay-e-commerce-integration) | E-commerce payment integration |

## Usage

Install a skill using the [Skills CLI](https://docs.agentmail.to/integrations/skills#getting-started):

```bash
npx skills add centrapay/centrapay-skills --skill centrapay-e-commerce-integration --agent claude-code
```

Or copy the contents of `SKILL.md` directly into your project context.
